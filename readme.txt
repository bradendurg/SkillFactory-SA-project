Представлены итоговая дипломная работа, а также презентация, которая использовалась на защите.

Идея проекта в дипломной работе - сервис онлайн перевода диалога.
В дипломе проводилась работа по анализу требований, описанию бизнес процессов, определялось взаимодействие с ЗС, описаны програмные интерфейсы и т.д.
Проведенная работа охватывает весь стандартный список компетенций для системного аналитика.

По итогам защиты получена высокая оценка.
